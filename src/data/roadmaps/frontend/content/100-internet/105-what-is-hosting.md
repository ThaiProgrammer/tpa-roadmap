# โฮสติ้ง ( Hosting )

การโฮสเว็บไซต์ คือ บริการออนไลน์ที่อนุญาติให้คุณขึ้นหน้าเว็บไซต์บนโลกอินเตอร์ได้ ดังนั้น ใครก็ตามที่สามารถเข้าถึงอินเตอร์เน็ตก็จะสามารถเข้าถึงเว็บไซต์ของคุณได้เช่นกัน
