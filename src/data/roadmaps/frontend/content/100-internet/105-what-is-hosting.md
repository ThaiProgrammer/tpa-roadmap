# Hosting

เว็บโฮสติ้ง (Web hosting) คือบริการออนไลน์ที่อนุญาตให้คุณเผยแพร่ไฟล์เว็บไซต์ของคุณบนอินเทอร์เน็ต ดังนั้น ทุกคนที่สามารถเข้าถึงอินเทอร์เน็ตได้ก็จะสามารถเข้าถึงเว็บไซต์ของคุณได้เช่นกัน
