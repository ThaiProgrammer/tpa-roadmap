# Javascript Versions

JavaScript ถูกสร้างโดย Brendan Eich และในปี ค.ศ. 1997 มันกลายเป็นมาตรฐาน ECMA โดยใช้ชื่ออย่างเป็นทางการว่า ECMAScript

ECMAScript มีหลาย Version ประกอบไปด้วย ES1, ES2, ES3, ES5 และ ES6
