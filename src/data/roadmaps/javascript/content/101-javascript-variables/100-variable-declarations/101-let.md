# [let] keyword

คำสั่ง let ใช้ประกาศตัวแปรเฉพาะที่ (local variable) ในขอบเขตแบบบล็อก (block-scoped) โดยตัวแปรสามารถถูกกำหนดค่าเริ่มต้นได้ตามต้องการ
