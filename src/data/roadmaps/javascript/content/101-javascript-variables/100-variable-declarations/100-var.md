# [var] keyword

คำสั่ง var ใช้ประกาศตัวแปรในขอบเขตฟังก์ชัน (function-scoped) หรือในขอบเขตส่วนกลาง (global scope) โดยตัวแปรสามารถถูกกำหนดค่าเริ่มต้นได้ตามต้องการ
