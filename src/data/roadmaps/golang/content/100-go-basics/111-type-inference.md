# Type Inference

Type inference ช่วยให้สามารถตรวจจับประเภทของค่าโดยได้ไม่ต้องระบุอย่างชัดเจน ดังนั้นจึงมีความเป็นไปได้ที่จะประกาศตัวแปรโดยไม่ต้องระบุประเภทในตอนแรกได้
