# Echo

Echo เป็นเฟรมเวิร์กแอปพลิเคชันเว็บของ Go แบบโอเพ่นซอร์สที่เน้นประสิทธิภาพ ขยายได้ เป็นเฟรมเวิร์กเว็บที่เรียบง่ายซึ่งอยู่ระหว่าง stdlib + router และเว็บเฟรมเวิร์กแบบ full-stack
