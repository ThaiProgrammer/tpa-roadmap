# Logging

Go มีคุณสมบัติในตัวเพื่อให้โปรแกรมเมอร์ใช้สามารถใช้งานการ log ได้ง่ายขึ้น โดยนักพัฒนาบุคคลที่สามยังได้สร้างเครื่องมือเพิ่มเติมเพื่อทำให้การ  log นั้นง่ายขึ้น
