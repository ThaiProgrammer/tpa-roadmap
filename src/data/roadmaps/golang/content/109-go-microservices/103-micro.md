# Micro

แพลตฟอร์มนี้เป็นแพลตฟอร์มที่พัฒนาด้วยแนวคิด API อย่างแรกเป็นหลัก โดยใช้รูปแบบการออกแบบที่เป็นแบบ microservices architecture และมีชุดของบริการที่ทำหน้าที่เหมือนกับบล็อกสำหรับการสร้างแพลตฟอร์ม
